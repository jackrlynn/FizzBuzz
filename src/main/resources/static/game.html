FizzBuzz Game

Rules

1. Count from 1 to as far as you go following the parameters.
2. If the number is divisible by 3, then select "Fizz".
3. If the number is divisible by 5, then select "Buzz".
4. If the number is divisible by 3 and divisible by 5, then select "FizzBuzz".
5. Otherwise, select its numerical value.
